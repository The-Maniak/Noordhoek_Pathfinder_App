UK crewlist - instructions
This feature allows you to get an UK format crewlist (FAL4) and crew effects declaraion sheet (FAL5) from vessel's format crewlist.

1. Select the NHP format crewlist you want take the data from.
2. Create the FAL4 and FAL5 files by pressing "Create an UK crew list" button
3. The generated files you can find in 'ready-documents' subdirectory.
4. You still need to check and update the port you're heading to, date,
arrival / departure etc.
